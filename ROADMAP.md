##  upcoming：v1.1.0 🚧

### 🎯 核心目标：智能监控策略优化

基于用户反馈和性能考虑，我们将优化文件监控策略，从"监控所有文件"转变为"智能监控"。

### 📋 功能列表

#### 1. 智能监控范围
- **监控搜索过的文件**：只监控用户曾经搜索过的文件
- **监控常用目录**：监控用户指定的常用工作目录
- **智能扩展**：根据用户使用习惯自动调整监控范围

#### 2. 自适应监控频率
- **高频监控**：对活跃文件（最近搜索/修改）进行高频监控
- **低频监控**：对不活跃文件降低监控频率
- **用户可配置**：允许用户自定义监控频率

#### 3. 监控白名单
- **用户自定义**：允许用户手动添加重要目录到监控列表
- **智能推荐**：根据文件访问频率推荐添加监控
- **批量管理**：支持批量添加/移除监控路径

#### 4. 性能优化
- **减少资源占用**：大幅降低监控的文件数量
- **降低 CPU 使用率**：减少不必要的文件系统扫描
- **内存优化**：优化监控状态存储

### 🔧 技术实现

```python
# 新的监控配置
class SmartMonitorConfig:
    # 监控模式
    mode: str = "smart"  # "smart", "full", "custom"
    
    # 智能监控设置
    monitor_searched_files: bool = True
    monitor_custom_dirs: List[str] = []
    
    # 自适应频率
    active_file_interval: int = 5  # 活跃文件监控间隔（秒）
    inactive_file_interval: int = 300  # 不活跃文件监控间隔（秒）
    
    # 白名单
    whitelist: List[str] = []
    blacklist: List[str] = []
```
